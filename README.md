# 静心自律 - 科学行为管理助手

> 基于认知行为疗法(CBT)、正念技术与神经可塑性科学，帮助你重塑行为模式、找回自由意志。

---

## 🧠 科学基础

本程序汲取以下科学领域的研究成果：

| 领域 | 核心理论 | 应用 |
|------|---------|------|
| **神经可塑性** | Hebb定律："一起放电的神经元会连接在一起" | 每一次克制都在物理层面重塑大脑回路 |
| **认知行为疗法(CBT)** | 想法→情绪→行为的三角模型 | 识别并挑战自动化思维 |
| **冲动冲浪** | Alan Marlatt的正念干预技术 | 观察冲动而非对抗，让其自然消退 |
| **习惯循环** | Charles Duhigg的暗示→行为→奖励模型 | 识别触发因素，替换惯常行为 |
| **自我监控效应** | 行为追踪即行为改善 | 日历热力图、连续天数计数 |
| **多巴胺戒断** | 奖励系统下调与上调机制 | 90天系统重置计划 |
| **自控力训练** | Baumeister的自我损耗理论 | 渐进式自控力增强 |

## 🚀 快速开始

### 方法一：直接打开（推荐）

1. 双击 `index.html` 即可在浏览器中打开
2. 所有数据保存在浏览器的 localStorage 中（本地，不会上传）
3. 建议将浏览器书签固定在标签栏

### 方法二：设为浏览器主页

将 `index.html` 的完整路径设为浏览器主页，每次打开浏览器都会启动。

### 方法三：开机自启动 ⭐

**自动配置（推荐）：**
1. 双击 `setup_startup.bat`（无需管理员权限）
2. 按提示完成即可
3. 下次开机自动启动

> 💡 如果双击失效，可右键 `setup_startup.ps1` → 使用 PowerShell 运行

**手动配置：**
1. `Win+R` → 输入 `shell:startup` → 回车
2. 将 `startup.vbs` 的**快捷方式**复制到打开的文件夹
3. 完成

## 📋 功能模块

### 📊 仪表盘
- 当前连续天数、累计自律天数、目标完成度、最佳记录
- 进度环可视化
- 里程碑追踪（3天→7天→14天→21天→30天→60天→90天→180天→365天）
- 每日励志金句

### ✅ 每日打卡
- 一键签到，记录自律成功
- 诚实记录破戒（关键！数据完整性）
- 每日反思日记

### 📅 记录日历
- 365天 GitHub 风格热力图
- 颜色深浅反映连续天数
- 红色标记破戒日

### ⏱️ 冲动冲浪计时器
- 正念观察冲动的科学方法
- 4-7-8呼吸引导（吸气4秒+屏息7秒+呼气8秒）
- 5/10/20/30分钟可选时长
- 完成自动记录

### 🆘 紧急应对
- 6种即时应对策略
- 冲动日志（记录触发因素+强度+感受）
- 一键跳转冲动冲浪

### 📚 心理学知识库
- 8篇核心心理学文章
- 涵盖神经可塑性、冲动冲浪、习惯循环、多巴胺戒断、自控力训练、SMART目标、CBT、自我监控

### 🏆 成就系统
- 16个成就徽章（从"第一步"到"一年传奇"）
- 自动解锁+庆祝动画

### 📈 数据分析
- 月度自律天数趋势图
- 成功率统计
- 最常见触发因素分析

## 💡 使用建议

### 第一周：建立基线
1. 诚实记录每一天（包括破戒日）
2. 填写冲动日志，识别触发模式
3. 阅读"神经可塑性与成瘾"和"冲动冲浪技术详解"

### 第二周：主动应对
1. 每次冲动来袭时使用冲动冲浪计时器
2. 实践4-7-8呼吸法
3. 制定"如果-那么"计划

### 第三周+：巩固新习惯
1. 保持每日打卡和日记
2. 回顾数据分析，识别高危情境
3. 调整应对策略

- 纯前端HTML/CSS/JS，无需服务器
- 数据存储在浏览器的 localStorage
- 支持所有现代浏览器（Chrome/Edge/Firefox/Safari）
- 响应式设计，支持手机和电脑
